# **운동 루틴 기록 서비스**

- 자신만의 운동 루틴을 만들어 손쉽게 기록하자
- 루틴 단위로 운동을 관리하여 요일별로 규칙적인 운동을 하자
- 회수 / 시간 기준으로 운동 세트를 만들 수 있다
- 매일 매일 운동을 하고 기록으로 남길 수 있다
